# 자유게시판(cafeboard) toy project

## 💻 cafeboard project
2023.08 - 2023.09

## 🙂 멤버
- kim tae hun

## 💼 프로젝트 소개
개발하면서 습득한 지식들을 잘 이용해서 누구나 자유롭게 어떤 주제로든 이야기가 가능한 게시판을 만들었습니다.
실제로 이번 프로젝트에서 운영과 유지보수를 진행했습니다.

## ⚙️ 개발 환경
- java 8
- jdk 1.8
- IDE : intellij community ver
- FRAMEWORK : springBoot(2.7.14)
- DATABASE : PostgreSQL
- ORM : Mybatis
